# MSKJ Pinturas
https://mskjarteedecoracoespinturas.netlify.app/

Sistema web desenvolvido para apresentação dos serviços da **MSKJ Pinturas**, empresa especializada em pinturas residenciais, comerciais e reformas em geral.

## Sobre

A MSKJ Pinturas oferece soluções profissionais em pintura, acabamento e revitalização de ambientes, priorizando qualidade, comprometimento e satisfação dos clientes.

## Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript

## Funcionalidades

* Página responsiva
* Design moderno e profissional
* Formulário de contato
* Apresentação de serviços
* Compatível com dispositivos móveis

## Como Executar

1. Clone este repositório:

```bash
git clone https://github.com/seu-usuario/mskj-pinturas.git
```

2. Abra o arquivo `index.html` em seu navegador.
